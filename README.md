---------|--------------|-------|----- 
userName | Y            | String|用户名

#### 请求方式
POST
#### 参数

参数名称      | 是否是必须的 | 类型  | 说明
--------------|--------------|-------|----- 
userName      | Y            | String|用户名
password      | Y            | String|密码 
code          | Y            | String|验证码(图片)
safetyQuestion| Y            | String|安全问题
safetyAnswer  | Y            | String|安全问题答案

</pre>
